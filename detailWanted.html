Imac Wanted

Native HTML 5, CSS and Javascript game created for a maths class. Imac Wanted is a Remake of a famous game where are included several mathematical functions to define the behavior of the characters. The player has to find the wanted character in the crowd by clicking on it. The crowd position, movement, and speed respect differentmathematical functions manually implemented. For instance, the probability that a character of the crowd appears respects a Markov chain that we have defined and coded.
