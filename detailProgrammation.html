BACK-END PORTFOLIO

This project realised in group of 4 is a portfolio based on a database. In fact, we have created a database of our previous projects, that we linked with a front end we develop ouselves with native HLML, CSS and Javaascript. This web site is devided in two parts : a visitor part and an adminstator part. In the visitor view, this website is a simple portfolio. Whereas after the connexion, the admiinistrator has the possibility to modify, delete, or add elements to the portfolio. By doing this the admin connects hiseflsef to the database and change it. This exchanges system was mostly done in PHP.
